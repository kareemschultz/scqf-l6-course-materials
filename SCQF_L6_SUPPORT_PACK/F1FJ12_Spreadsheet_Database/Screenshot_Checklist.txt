  F1FJ12 - SPREADSHEETS AND DATABASES: SCREENSHOT CHECKLIST
================================================================================

  Capture the following screenshots and embed them in your report document.
  Label each one clearly with a Figure number and description.

  TIP: Use the Snipping Tool (Win + Shift + S) or Print Screen to capture.
  Paste into your report and add a caption below each image.

================================================================================
  TASK A - EXCEL SCREENSHOTS
================================================================================

  [ ] 1. Filtered data view
      Show the spreadsheet with filters applied (filter arrows visible
      in column headers, filtered rows displayed).

  [ ] 2. Column chart
      Show the completed column chart with axis labels, title, and legend.

  [ ] 3. Pivot table
      Show the pivot table with data summarised by your chosen fields.

  [ ] 4. Slicer
      Show the slicer panel connected to your pivot table, with at least
      one item selected to demonstrate filtering.

  [ ] 5. Data validation dropdown
      Show the cell with the dropdown arrow visible, AND show the dropdown
      list expanded (click the arrow to show the options).

  [ ] 6. Dynamic summary area
      Show the summary formulas (e.g. SUMIFS, COUNTIFS) calculating results
      based on the dropdown selection.

  [ ] 7. Formula view (Ctrl + `)
      Press Ctrl + ` (grave accent) to switch to formula view. Capture the
      sheet showing actual formulas instead of values. Make sure columns are
      wide enough that formulas are not cut off.

  [ ] 8. VBA Macro code
      Open the Visual Basic Editor (Alt + F11). Show the macro code in the
      code window. The macro name and all lines of code should be visible.

  [ ] 9. Macro result (before and after)
      Show the data BEFORE running the macro, then show it AFTER running
      the macro so the formatting changes are visible.

  [ ] 10. VLOOKUP formula and result
       Show the cell containing your VLOOKUP formula in the formula bar,
       AND show the returned result in the cell.

================================================================================
  GENERAL TIPS
================================================================================

  - Make sure every screenshot is large enough to read. If text is too small,
    zoom in before capturing.
  - Use consistent figure numbering: Figure 1.1.1a, Figure 1.1.1b, etc.
  - Add a one-sentence caption below each figure explaining what it shows.
  - Do NOT include screenshots of you browsing the internet or opening files.
    Only include screenshots that demonstrate the required technical features.

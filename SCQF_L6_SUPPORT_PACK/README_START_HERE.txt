================================================================================
  WHAT IS IN THIS PACK
================================================================================

  This folder contains resources for all five units:

  1. F1FJ12 - Spreadsheets and Databases (70 marks)
  2. F1FE12 - Word Processing and Presenting (70 marks)
  3. J22976 - Understanding Business (100 marks)
  4. J22A76 - Management of People and Finance (100 marks)
  5. HE9E46 - Contemporary Business Issues (100 marks)

  Each subfolder contains:
  - Blank templates with section headings and placeholder text
  - Sample datasets for practice (using generic dummy data)
  - Technical guides explaining HOW to use required software features
  - Checklists telling you what evidence and screenshots to capture

  UNIT OVERVIEWS

  UNIT: F1FJ12 - Spreadsheets and Databases (70 marks)
  ----------------------------------------------------
  Format: Excel workbook (.xlsm) + Access database (.accdb) + Report (PDF)

  Task A (40 marks):
  - Filter and sort data in Excel
  - Create column charts and pivot tables with slicers
  - Set up data validation dropdowns and dynamic summary formulas
  - Record and run a VBA macro to format a report
  - Use VLOOKUP to retrieve data from a lookup range
  - Build forms and queries in Microsoft Access

  Task B (30 marks):
  - Justify why Excel is suitable for business reporting
  - Create additional charts with trendlines, R-squared values, and forecasts
  - Build accountant-style tables with Profit formulas, SUMIF, and IF functions
  - Justify why Access is suitable for data entry forms
  - Build a patient treatment entry form and summary query
  - Evaluate strengths and limitations of both Excel and Access

  MANDATORY TECHNICAL FEATURES:
  [x] Filtered data views
  [x] Column chart and line chart with trendline + equation + R-squared
  [x] Pivot table with at least one slicer
  [x] Data validation dropdown list
  [x] Dynamic summary using SUMIFS / COUNTIFS
  [x] VBA macro (recorded or written)
  [x] VLOOKUP formula
  [x] Access relationships between tables
  [x] Access form with listbox and command buttons
  [x] Access query (sorted, and a join query)

  MANDATORY SCREENSHOTS:
  See Screenshot_Checklist.txt in the F1FJ12 folder.


  UNIT: F1FE12 - Word Processing and Presenting (70 marks)
  --------------------------------------------------------
  Format: Word document + PowerPoint presentation + Report (PDF)

  Task A (50 marks):
  - Create a 4-page conference document with headers, footers, Heading styles
  - Insert an automatic Table of Contents
  - Perform a mail merge to create delegate ID badges
  - Insert a logo with text wrapping and a formatted agenda table
  - Build a 5-slide presentation with action buttons and navigation
  - Embed video (auto-play) and audio on slides
  - Configure kiosk mode (self-running loop) and copyright citation

  Task B (20 marks):
  - Justify why Word was chosen over PowerPoint for the document (300+ words)
  - Review efficiency: identify a repetitive task and suggest automation (200+ words)
  - Review audience: compare kiosk mode vs keynote delivery (200+ words)

  MANDATORY TECHNICAL FEATURES:
  [x] Heading 1 / Heading 2 styles applied consistently
  [x] Automatic Table of Contents
  [x] Mail merge from an external data source (e.g. Excel)
  [x] Image with text wrapping (e.g. Tight or Square)
  [x] Formatted table with specific column widths and shading
  [x] Action buttons on slides (navigation to other slides)
  [x] Embedded video with auto-play setting
  [x] Embedded or linked audio
  [x] Kiosk mode (loop until Esc)
  [x] Copyright citation on final slide


  UNIT: J22976 - Understanding Business (100 marks)
  --------------------------------------------------
  Format: Report document (PDF)

  Task A:
  - Compare three types of organisation (different sectors/scales)
  - Compare types of business ownership
  - Explain organic vs inorganic growth with examples
  - Analyse business objectives and why they were adopted
  - Explain organisational structures with real examples
  - Explain strategic, tactical, and operational decisions

  Task B:
  - Analyse internal factors affecting business performance
  - Analyse external factors using PESTEC framework
  - Analyse stakeholders, their influence, and conflicts

  MANDATORY FEATURES:
  [x] Comparative analysis (not just describing one organisation at a time)
  [x] Harvard in-text citations and a reference list
  [x] Mapping table linking each task to marking criteria
  [x] Cover page with student details
  [x] Declaration of Originality
  [x] Automatic Table of Contents


  UNIT: J22A76 - Management of People and Finance (100 marks)

  MANDATORY FEATURES:

  [x] Harvard in-text citations and a reference list
  [x] Mapping table
  [x] Cover page and Declaration of Originality

================================================================================
  IMPORTANT WARNINGS
================================================================================

  1. DO NOT copy text from any other student's work, online essay banks, or
     AI-generated content. Your college uses plagiarism detection software.

  2. USE YOUR OWN ORGANISATIONS. Choose businesses you are familiar with or
     have researched yourself. Do not use the same organisations as your
     classmates.

  3. WRITE IN YOUR OWN WORDS. Even when explaining theories (like Maslow or
     PESTEC), put the explanation into your own language. Paraphrasing from
     textbooks is fine; copying sentences is not.

  4. KEEP YOUR TONE at SCQF Level 6. You are expected to show understanding
     and basic evaluation. You do not need to write like a university
     dissertation.

  5. CHECK YOUR BRIEF CAREFULLY. The marking criteria table in your brief
     tells you exactly what the assessor will look for in each section.
